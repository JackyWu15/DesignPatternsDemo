# DesignPatternsDemo

- 这个Demo是通过对<<android设计模式和源码分析>>一书阅读的练习和总结,对每个设计模式都做了关键点的注释和概括

- 每个设计模式在Android Frameworks的源码设计中都有具体的使用，通过查看分析源码，有利于加深对设计模式的理解，Demo在每个模式的开头，都标注了源码对应使用的场景。
